# airbnb-clone

airbnb clone by Ivan Popov.

# Front-end Technologies:

- Next.js
- TypeScript
- SCSS (Sass)
- Zustand (state management)
- React Query
- React Hook Form
- Next Auth (authentication)
- Zod (validation)
- Axios (HTTP requests)

# Back-end Technologies:

- NestJS
- Prisma ORM
- CASL (permissions)
- JWT (authentication)
- Argon2 (password hashing)
- Class Transformer
- Passport
